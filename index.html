Multi-Lingual Input Tools

Multi-Lingual Input Tools is a browser extension available on the Chrome Webstore and Microsoft Edge Add-ons.
It helps people copy and paste accented letters and symbols from a growing list of languages. So far the avilable languages are Spanish (v0.0), French (v5.0), and German (v6.0).
So far, the accented letters and symbols that can be copied are...

  • Spanish: á, Á, é, É, í, Í, ó, Ó, ú, Ú, ¿, ¡
  • French: à, À, â, Â, æ, Æ, ç, Ç, é, É, è, È, ê, Ê, ë, Ë, ï, Ï, î, Î, ô, Ô, œ, Œ
  • German: ä, Ä, ö, Ö, ü, Ü, ß, ẞ
